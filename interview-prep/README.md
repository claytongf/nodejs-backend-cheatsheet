## Recommended Practice Loop

1. Pick one topic.
2. Answer without reading the answer.
3. Compare against the strong-answer signals.
4. Point to a real file in this repository.
5. Improve the answer in one sentence.
6. Repeat the answer out loud.

Good interview preparation is not memorizing lines. It is being able to explain a real
decision, trade-off, or bug clearly under pressure.
